# 📄 Resume Job Analyzer

A modern NLP-powered web application that analyzes how well a resume matches a job description and provides actionable feedback for improving ATS compatibility.

## 🚀 Features

- Upload Resume PDF
- Paste Job Description
- Calculate Resume-Job Match Score
- Identify Matched Keywords
- Detect Missing Keywords
- Generate Resume Improvement Suggestions
- Interactive and User-Friendly UI

## 🛠️ Technologies Used

- Python
- Streamlit
- PyPDF2
- Scikit-Learn
- TF-IDF Vectorization
- Cosine Similarity

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/azafarbutt/resume-job-analyzer.git
```

Move into the project directory:

```bash
cd resume-job-analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
